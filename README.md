# TechGearShop

An e-commerce platform for technology products built with MVC architecture.

## Technologies Used

- **Backend**: C# / ASP.NET MVC
- **Frontend**: HTML, CSS/SCSS, JavaScript
- **Admin Template**: Purple Admin (Bootstrap 5)
- **Architecture**: Model-View-Controller (MVC)
